安裝與準備

先準備一個 Codex 專用資料夾

新手建議使用 Codex 桌面 App。登入後，選擇一個專案資料夾，讓 Codex 只在這個資料夾內讀取與處理檔案。

安裝步驟

  • 1準備 ChatGPT 帳號，開啟 Codex App 或 Codex 支援的工具。
  • 2建立資料夾，例如「電子報報表分析」。
  • 3把 InfoCenter 匯出的成效報表、連結點擊分析、寄送明細放進同一個資料夾。
  • 4在 Codex 選擇這個資料夾，權限先維持預設。
  • 5先請 Codex 只檢查與分析，不要修改檔案。

使用流程

從 InfoCenter 到 Codex 的五步驟

把重點想成一條路線：先匯出，再集中，再讓 Codex 讀懂，最後把洞察變成下一次電子報的改善清單。

1

在 InfoCenter 製作電子報

設定區塊、內容、社群連結、排列方式與版面樣式，依活動或公告需求調整。

2

寄送後下載報表

下載開信率、點擊率、送達率、退信率、投訴紀錄、ISP 分布與寄送明細。

3

同一活動放同一資料夾

把相關 Excel、CSV 或報表檔集中，檔名加上日期與活動名稱，Codex 會更容易理解。

4

請 Codex 分析

用新手提示詞請 Codex 找出熱門連結、異常指標、主旨與內容區塊表現。

5

轉成下次改善清單

把分析結果整理成下一次電子報的主旨、版面、CTA 與寄送策略建議。

報表重點

請 Codex 優先看這些指標

不用先學資料分析術語。先讓 Codex 把數字翻成「這代表什麼」與「下一步怎麼做」。

主旨吸引力 開信率

判斷收件者是否願意打開信件，可比較不同主旨或寄送時間。

內容吸引力 點擊率

判斷內容是否讓人有下一步行動，可搭配連結點擊排行分析。

寄送品質 送達率

觀察信件是否成功送到收件端，搭配 ISP 分布可找出特定服務問題。

風險訊號 退信/投訴

若比例異常，應檢查名單品質、內容適切性或寄送設定。

安全守則

              
新手使用 Codex 的三個原則

            

            

              Codex 很適合協助分析，但仍要注意資料權限與個資。最穩的做法是先分析、再確認、最後才修改或輸出新檔。
            

          


          

            

              
只用專案資料夾

              
把要分析的報表集中在一個資料夾，讓 Codex 只處理這個範圍內的檔案。

            

            

              
先要求不修改

              
第一次請 Codex「只讀取與分析，不要修改檔案」，等理解結果後再決定下一步。

            

            

              
留意個資與權限

              
若報表含 Email、電話或個人資訊，先確認內部資料使用規範，再交給 Codex 分析。

            

          


          

            建議第一句話就寫清楚：「我沒有 IT 或資料分析背景，請用管理者看得懂的方式說明，並把每個數字代表的意義講清楚。」
